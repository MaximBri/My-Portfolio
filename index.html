Кто же такой Frontend-разработчик???

Если коротко, то этот человек похож на того типа снизу

Но в глобальном смысле фронтенд-разработчик (frontend developer, фронтендер) — это специалист, который создаёт интерфейсы для сайтов и веб-приложений. Он отвечает за то, с чем пользователи взаимодействуют на страницах: текст, изображения, анимации, кнопки, формы, слайдеры, галереи и прочие «красивости». Ещё фронтендер делает так, чтобы сайт одинаково хорошо работал на всех устройствах.

HTML 5

Для вёрстки

Изучен недавно

Язык гипертекстовой разметки текста. Он нужен, чтобы размещать на веб-странице элементы: текст, картинки, таблицы и видео.

CSS 3

Для вёрстки

Язык таблиц стилей, который позволяет прикреплять стиль (например, шрифты и цвет) к структурированным документам

JavaScript

Для вёрстки

JavaScript — это язык программирования, который позволяет делать веб-приложения динамичными и интерактивными.

jQuery

Для вёрстки

jQuery — это библиотека для языка программирования JavaScript, набор инструментов для веб-разработки.

React

Для вёрстки

React — это библиотека JavaScript с открытым кодом для создания внешних пользовательских интерфейсов.

Sass/Scss

Для ускорения вёрстки

Метаязык, который упрощает и ускоряет написание CSS-кода. Sass имеет свой синтаксис, из которого генерирует CSS-код, понятный любому браузеру.

Git/Github

Для сохранений

GitHub — это веб-сайт, на котором можно хранить проекты Git. Git — это средство отслеживания изменений, а GitHub — это место для хранения проектов и совместной работы.

Figma

Для просмотра макета

С макета, разработанным дизайнером, веб-разработчик берёт все детали сайта (тексты, стили, шрифты, отступы) и воссоздаёт его копию в виде кода, который можно использовать для посадки на CMS (система управления контентом)

WordPress

Система управления контентом

Система управления контентом веб-ресурсов (CMS, Content Management System), которая является одной из самых востребованных по всему миру.

ACF

Плагин для CMS WordPress

Позволяет создавать поля в админ-панели сайта, благодаря чему можно поменять тексты или изображения на сайте даже неопытному человеку без помощи веб-разработчика

Ответы на популярные вопросы

  • Что такое вёрстка?

    Вёрстка — процесс создания структуры кода по отрисованному макету, как правило используя HTML, CSS, JavaScript, а также прочие инструменты для ускорения процесса. Результатом вёрстки является папка, внутри которой находится запускаемый в браузерах файл с расширением html, а также другие папки с данными, шрифтами, CSS и JavaScript кодом. Человек, делающий эту работу, называется верстальщиком.

  • Что такое адаптивность?

    Адаптивность означает, что сайт одинаково хорошо отображается как на компьюторе, так и на телефонах и планшетах. При этом контент сайта может отличаться в зависимости от устройства: на телефонах в шапке сайта достаточно часто реализуют бургер-меню, при нажатии на которое открывается панель с дополнительным контентом или информацией

  • Что такое слайдер?

    Слайдер представляет собой набор связанных элементов, которые имеют некоторый контент внутри. При этом пользователю показывается только часть элементов, остальные спрятаны и появляются при определённом действии со стороны пользователя. Пример слайдера реализован в блоке "Мои навыки"

  • Что такое миксер?

    Миксер - набор элементов с кнопками, при нажатии на которые показываются некоторые элементы (возможно все). Кнопки служат "фильтрами" для элементов. Пример миксера реализован в блоке "Портфолио"

  • Что такое аккордеон?

    Аккордеон - список элементов, каждый из которых содержит скрытый контент, который показывается при клике на определённую кнопку или весь элемент вцелом. Примером аккордеона служит этот блок

  • Что такое лендинг?

    Лендинг – это самостоятельный сайт или отдельная страница, призывающий посетителей совершить целевое действие.
